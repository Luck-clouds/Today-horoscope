\### 功能介绍

1\. 分发一个api用于获取今日运势

2\. 分发一个api可以看到服务器运行情况

3\. 多次请求封禁

4\. 可以看到万年历 年、月、日 二十四节气

5\. 主机日志打印 时间 ip 请求方式 路径 状态 请求所需时间
